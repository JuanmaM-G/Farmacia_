Objetivo:
El objetivo del proyecto fue optimizar la gestión de inventario farmacéutico mediante un sistema de control en tiempo real de existencias, lotes y fechas de vencimiento.


Tecnologias usadas:
Python
React
Vite
JavaScript
Sql


Pasos para ejecutar:
1- Ejecutar "Xampp" tanto en Apache como en MySQL
    1.2- Espichar el boton de "admin" en MySQL
    1.3- Crear la base de datos o importar el archivo "schema.sql" en phpmyadmin
2- Ubicarnos en el archivo "app.py"
    2.1- Oprimir la tecla "F5", seleccionar la opcion de "python debugger", despues seleccionar la opcion de "python file"
3- Ubicarnos en el archivo "App.jsx"
    3.1- Abrir la terminal de visual con la combinacion de "CTRL + Ñ"
    3.2- Ubicarnos en la carpeta que contenga la carpeta "src", usando el comando "cd retardos"
    3.3- poner el comando "npm run dev"
    3.4- Poner la siguiente url "http://localhost:5173" en un navegador web
